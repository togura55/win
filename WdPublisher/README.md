# WdPublisher
## Description
Socket client application for reading raw data of WILL devices and transferring to WdBroker  
## Build Instruction
1. Load sln by Visual Studio 2017  
2. Add references of WILL Ink 2.1, WILL Devices 1.0 and WILL InkCanvas UWP  
3. Build it
