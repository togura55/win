<Description>
This tool allows user setting and displaying the upper 6-digit of barcode value which CLB Create is configured at present. A user can choose one of the UI mode from GUI or CLI.

<Instruction>
Double click the application icon on the desktop OR run the application with command-line swiches on the command prompt window.

<Requirement>
Windows 7 and later with .NET Framework 4.0

<Comman-line Parameters>
-ui:gui|cli "UI mode is GUI(default) | CLI"
-bc:XXX "Specify upper 6-digit barcode"
-bi:XXX "Increment/decrement barcode value. X|-X"
-td:XXX "Specify the directory of CLB template folder"
-cv:XXX "Specify the version/folder number of CLB Create config"
-sp:XXX "Specify the directory path of CLB Create config"
-h:on|off "Display help contents | invisible(default)"
-v:on|off "Display version number | invisible(default)"
