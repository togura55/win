<説明>
この配布プログムでは、Wacom Ink SDK for signatureライブラリとライブラリを使った電子サインキャプチャーアプリがPCにインストールされます。

<インストールとアプリの実行>
- 管理者権限でコマンドプロンプトを実行する
- コマンドプロンプト上でinstall.batを実行する
- WacomサインデバイスをPCに接続されていることを確認する
- WindowsメニューからTestSigCaptを実行
- [Sign]ボタンをクリックして、電子サインを記入
- サインデバイス上の[OK]をタップ

<補足>
取得された電子サイン画像は、TestSigCapt.exeのインストール先（既定ではC:\Program Files (x86)\Wacom\TestSigCapt\）にPNG形式で保存されます。

<バージョン>
SDK: Wacom-Signature-SDK-x64-4.5.5
App: TestSigCapt.exe
OS: Windows 10
Device: Wacomサインデバイス (STU, DTU, DTKシリーズ）
